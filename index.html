Pip producers is a blog site for currency and day traders.

Pip producers is built on the MERN stack architecture. MongoDB, Express, React, Node js

Initial Setup

If you are not running the application locally you can skip to "Inside the Application"


Setup prerequisits:

  • Internet connection
  • Node js and NPM installed
  • Terminal access
  • Basic knowledge of each item

1) The project can be cloned directly from the git repository or downloaded.

2) Run the following scripts from the root directory to installdependencies:

                npm install
                npm run client-install

***the scripts above will work with yarn

To connect to a database the URI should be placed within a file called default.json and placed inside theconfig folder of the root directory. The project will work with any Mongo shell i.e. RoboMongo or mlab

The database can be named anything as long as the DB URI contains the correct title

Inside of default.json

                    {
                        'mongoURI": "mongodb://<username>:<dbpassword>@<dsreference>.mlab.com:31207/pip_producer",
                        "JWT_SECRET": "<secret random string here>"
                    }

3) Use the following scripts to run the application:

                Run server on port 5000, and client on port 3000
                npm run dev
                

                Only start server on port 5000
                npm start
                

                Only start server with watch on port 5000
                npm run server
                

                Only run client (port 3000) without server
                npm run client

***the scripts above will work with yarn
